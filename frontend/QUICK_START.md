# 🚀 前端快速启动指南

## 📋 前置要求

确保已安装：
- Node.js 16+ 
- npm 或 yarn

## ⚡ 三步启动

### 1️⃣ 安装依赖

```bash
cd frontend
npm install
```

### 2️⃣ 启动开发服务器

```bash
npm run dev
```

### 3️⃣ 访问系统

浏览器打开：**http://localhost:3000**

会自动跳转到登录页面。

## ❓ 常见问题

### Q1: 登录后看不到某些菜单？
A: 检查当前角色，不同角色看到的菜单不同。

### Q2: WebSocket 连接失败？
A: 确保 notification-service 已启动（端口 8086）。

### Q3: API 请求失败？
A: 检查后端服务是否全部启动，查看 vite.config.js 代理配置。

### Q4: 如何切换角色？
A: 点击右上角"登出"，重新登录时选择不同角色。

## 🛠️ 开发提示

### 添加新页面

1. 在 `src/views/` 创建 Vue 组件
2. 在 `src/router/index.js` 添加路由
3. 在 `src/stores/user.js` 配置角色权限

### 修改菜单

编辑 `src/stores/user.js` 中的 `getMenuByRole()` 函数。

### 调整权限

在路由配置中使用 `meta.roles` 数组控制访问权限。

## 📞 技术支持

如有问题，请检查：
1. 浏览器控制台是否有错误
2. 后端服务是否正常运行
3. Redis 和 MySQL 是否启动
4. package.json 依赖是否正确安装
